# Tecnologías Utilizadas

Este proyecto utiliza un stack moderno de desarrollo web para garantizar velocidad, seguridad y una experiencia de usuario premium.

## Frontend
- **React (Vite)**: Framework principal para una interfaz rápida y reactiva.
- **Lucide React**: Biblioteca de iconos minimalistas y elegantes.
- **React ApexCharts**: Motor de visualización para gráficos interactivos (donas, áreas, líneas).
- **Date-fns**: Biblioteca avanzada para el manejo y formateo de fechas.
- **Vanilla CSS**: Sistema de estilos personalizado para lograr el diseño Glassmorphism sin dependencias pesadas.

## Backend (Google Workspace)
- **Google Sheets**: Base de datos flexible y accesible.
- **Google Apps Script**: API intermedia que convierte los datos del Sheet en formato JSON para el Dashboard.
- **Google Drive API**: Utilizada para la resolución de imágenes y fotos adjuntas a los tickets.

## Herramientas de Desarrollo
- **ESLint**: Para asegurar la calidad y consistencia del código.
- **Git**: Control de versiones.
