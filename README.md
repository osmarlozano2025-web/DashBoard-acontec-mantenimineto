# APPTICKET - Dashboard de Gestión de Tickets

Este proyecto es un Dashboard de Control premium diseñado para el monitoreo en tiempo real de tickets de mantenimiento, obras y servicios. La plataforma permite centralizar la información proveniente de Google Sheets, ofreciendo una visualización clara del estado operativo y administrativo de cada tarea.

## Propósito
El objetivo principal de APPTICKET es facilitar la toma de decisiones mediante:
- **Visualización de Estadísticas**: Resumen de tickets nuevos, en proceso y terminados.
- **Seguimiento Administrativo**: Monitoreo de facturación, grupos de trabajo y distribución de tareas.
- **Filtrado Avanzado**: Búsqueda por mes, estado, tienda, localidad y provincia.
- **Acceso Multiusuario**: Sistema de login seguro para diferentes niveles de administración.

## Características Principales
- **Conectividad Total**: Sincronización automática con Google Sheets mediante Google Apps Script.
- **Diseño Glassmorphism**: Interfaz moderna, oscura y elegante con efectos de transparencia.
- **Gráficos Dinámicos**: Análisis visual de tendencias semanales y distribución por grupos.
- **Detalle de Tickets**: Vista profunda de cada ticket, incluyendo cronología de fechas y comentarios.
